User Defined Types

Unions

The union keyword allows you to create a variable that can be interpreted multiple ways. One variable is "unioned" with two or more types, in order for the variable to read or written as any one of those types. The data stored in memory is the same, but the different data types allow the binary data to be "filtered" or interpreted in different ways. 


    union CharType {
	char asciiData;
	uint8_t intData;
};

int main(void)
{
	union CharType firstChar;	
		
	firstChar.asciiData = "a";
    firstChar.intData += 1;
    //firstChar.asciiData = "b"
	firstChar.intData += 32;
	//firstChar.asciiData = "B"	
		
	return 0;
}

For example. A "char" data type takes up a byte in memory. A "char" variable could be unioned with a uint8_t, allowing it to also be used for integer operations.
