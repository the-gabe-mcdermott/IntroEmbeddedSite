Logic Levels

When the Arduino reads a pin, it looks for a voltage Logic Level associated with either a "1" or a "0", a "high", or a "low". Anything outside these boundaries will read back with an uncertain value.

Data Sheet Table 32.2: Common DC Characteristics

Symbol Name Minimum Maximum
Vh Voltage Input Low -0.5 V 0.3 V
Vl Voltage Input High 0.7 V Vcc + 0.5 V

So when wiring the button for the microcontroller to read, it important that the button open and closed states each feed a different voltage logic level to the input pin.
