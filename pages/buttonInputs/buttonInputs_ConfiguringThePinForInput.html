Configuring the Pin for Input

In order to configure a pin to be used for input on the Arduino, the pin direction must be set to "0" in that pin's direction register. Consider the following example that sets up pin 11 on the Arduino, and then reads it. 


  //Arduino Pin 11 Maps to PORTB bit 3.
  uint8_t directionRegData = DDRB;
  
  //Set the 3rd bit to "0" for input.
  directionRegData &= !0x03;
  
  DDRB = directionRegData;

  //Now read the input once.
  uint8_t inputRegData = PINB;

  //Mask out all bits of read data
  //except the 3rd bit.
  uint8_t currentButtonState = inputRegData &= 0x03;
