Wiring the Button For Input

In order to ensure either a logic level "1" or "0" is always supplied to the input pin, the button is wired as indicated in this diagram.

When the button is open, a voltage close to VCC (Minus whatever is spent on the resistor) is read on the input pin. The circuit will behave as modeled in this diagram.

When the button is closed, the alternate path to ground is introduced. Since the Input pin will be essentially connected to the circuit at ground, the voltage read in will be GND, or voltage "0". Well within the voltage low boundary.
