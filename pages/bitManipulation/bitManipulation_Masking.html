Bit Manipulation

Masking

Addresses let you read data into and out of memory. However, you are required to read or write all the bits at the address at once.

How do you set or clear and individual bit stored in a whole byte of memory?

The trick is to use a mask and abitwise operator to "select" for an individual bit.

Using a Mask

A mask is a variable (usually a constant). The variable holds a number that when read in binary, has specific bit positions set, in order to "select" those bits. 


uint8_t const firstWriteMask = 0x02;
//Here is the above mask shown in binary.
//firstWriteMask = 0000 0010


//Masks can also be used to select 
//multiple bits at a time.
//This mask "selects" the first bit
//And the last four bits.
uint8_t const secondWriteMask = 0xF1;
//Here is its representation in binary
//secondWriteMask = 1111 0001

Each bit that is set to a "1" in these masks will be used with a bitwise operation to "select" or mask that equivalent bit position in another variable.

Say you wanted to set the second bit of some number to a 1. You would use a mask to select for the second bit (0x02), and perform an AND operation on the number to set it to a 1.

Number 1111 0110
Mask 0000 0010
AND
Result 0000 0010

Writing to A Single Bit of Memory

A mask can be used with a bitwise operator (more on those in the next section), to set a specific bit in a variable. In order to set a bit in memory, use these steps.

  1. Read the value at the memory location you are interested in. Store the value in some temporary variable.
  2. Perform a bitwise operation on the value and a mask. The mask will specify the bit or bits you want to change. The bitwise operation will change those bits while leaving the rest of the bits at their previous value.
  3. Write the variable back to the memory location.

There are several bitwise operators you can use with this process to fidget with bits and get them to the exact values you want.
