Bit Manipulation

"There are 10 kinds of people in the world: those who understand binary numerals, and those who don't." - Ian Stewart

In embedded systems, a single bit in memory can be mapped to control the output of a single pin. A single bit may be used to control when a hardware timer starts, or how an interrupt behaves.

Point being, there will be times when you will need to toggle the state of an individual bit stored in memory.

However, most microcontrollers do not let you write the value of one bit in memory at a time. Groups of bits are read and overwritten together, usually in groups of 8, 16, or 32 bits at a time.

In this section, you will learn

  1. How addresses work, and how to use them to read or write a group of bits in memory.
  2. What a mask is, and how it is used to "select" individual bits out of a group of bits.
  3. What the different C Bitwise Operators are, and how they are used with masks to set or clear individual bits in a group of bits.
  4. How to apply these concepts to set or clear a bit in a C code example.
