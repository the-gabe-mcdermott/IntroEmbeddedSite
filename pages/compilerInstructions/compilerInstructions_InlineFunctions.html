Inline Functions

The inline keyword is used with a function declaration. It is a way to politely ask the compiler to treat the function as a macro.

Inline is A Request

Since the inline keyword is handled by the compiler and not by the preprocessor like with a #define macro, it behaves a little differently.

For one thing, error handling and type checking will work normally for code inside the inlined function.
