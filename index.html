Introduction to Embedded Systems
How to blink lights, read sensors, and other #justmicrocontrollerthings

Abstract

A guide to embedded system programming and design, with a focus on programming for the Aurduino Uno in the C programming language.

Senior Project of Gabriel McDermott for the Oregon Institute of Technology.

This guide assumes you know basic programming in C++ and simple computer architecture.
