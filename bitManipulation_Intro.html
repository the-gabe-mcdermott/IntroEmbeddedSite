Bit Manipulation

Bit by Bit

In embedded systems, a single bit in memory can be mapped to control the output of a single pin. A single bit may be used to control when a hardware timer starts, or how an intterupt behaves. However, most microcontrollers do not let you write the value of one bit in memory at a time. Groups of bits are read and overwritten together, usually in groups of 8, 16, or 32 bits at a time.
