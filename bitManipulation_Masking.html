Bit Manipulation

Masking

When I start writing values into memory, I use an addresses to specify where I want the data written. However, I have no way of writing a value to a single bit at an address. It’s kind of like how an apartment number can let me find a specific apartment to deliver a package to, but not a specific room in that apartment. This makes it impossible to write a single bit to say, turn on a system timer, without writing to the other bits at that address (which may control other things I don’t want to mess with.) The trick to get around this can be described in three steps.

  1. Read the value at the memory location you are interested in. Store the value in some temporary variable.
  2. Perform a bitwise operation on the value and a mask . The mask will specify the bit or bits you want to change. The bitwise operation will change those bits while leaving the rest of the bits at their previous value.
  3. Write the variable back to the memory location.
